Команды меню "Файл"

Меню "Файл" предоставляет команду "Октрыть таблицу", чтобы выбрать таблицу для выгрузки из БД и команду "Выход", чтобы завершить работу приложения.

Поднобнее о меню "Команда"        Команды для управления информацией в БД, так же есть на панели инструментов под главным меню.
