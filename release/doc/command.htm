Меню "Команда"

Ипользуйте команды из этого меню, чтобы управлять информацией в таблике БД. каждая команда проводит соответствующее действие над данными в БД. 

Для того, чтобы команды выполнялись нужно открыть таблицу (Файл - Открыть таблицу - ). Затем выделить желаемую строку в таблице и выполнить команду.
